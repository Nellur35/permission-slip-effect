# Experiments

Empirical testing of the reasoning pipeline and methodology components.

| Experiment | What it tests | Key finding |
|-----------|--------------|-------------|
| [Model Shootout](model-shootout.md) | Which Bedrock models perform best in multi-role reasoning pipelines | Switching from Llama 4 + DeepSeek to Kimi K2.5 + Mistral Large 3 shifted unique insight distribution from 13/1/1 (single-model dominance) to 3/2/2 (balanced). Model selection matters as much as pipeline design. |
